# ClaimDefenseAI

This project was made for the Atlanta Divergent Teams Hackathon (2026). Appealing medical insurance denial letters costs billions of dollars each year.
LLMs can be used to speed this process up, but they often hallucinate. Thus, this project is able to quickly load documents from popular insurers via a webscraper
and the included downloader script. They can be loaded into the vector database ChromaDB and quickly queried by the LLM to prevent hallucinations.
